# Phase 1 Ideation Expansion Prompt

Generate broad ideas before narrowing. Do not execute or propose code changes.

## Task
1) Generate about 100 ideas related to the problem.
2) Select the top 10 with a one-line rationale each and note value/effort/risk.
3) If new directions still appear, repeat the 100→top10 cycle once more.

## Output Format

### Idea List (Broad)
- 100 concise ideas (short phrases).

### Top 10 Selection
- Rank 1 to 10 with rationale.
- Include value/effort/risk tags for each.

### Recommended Directions
- 3 to 5 directions to consider for the plan update.

## Guardrails
- Keep ideas scoped to the problem.
- Avoid flashy features with unclear value.
- Do not add execution steps.
